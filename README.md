## WELCOME TO VELORA!!!

Hey! daswitez, here. This is a real state project that it's primary goal is to build and personalise the agency experience for the user. To achive this, I created Velora(I took inspiration from the internet, but I tried to make it unique and special for the project). I wanted to be creative and innovative, but also practical and functional, So we are going to take "Velora" to the next level. Many real state websites are good... but they don't really connect with clients, because let's be honest, they look very generic, but when you operate in a market like Europe, you need to be more sophisticated, you are not just designing for a country, you are designing for a lifestyle, for a culture, for a way of living (for example, the way of living in the Netherlands is different from the way of living in Italy and the weather is different too, so we need to adapt the design to every country).

## Documentation guide

Since we are adapting the UI, aesthetics, and colors to 8 different European countries, the project can get pretty big! To make your life easier, I've split everything into super easy-to-read folders inside the `docs/` directory:

- **`/docs/velora_concept_and_experience.md`**: Start here! It explains the whole vibe and logic behind the brand (the "Main Hub" vs the "Country Embassies").
- **`/docs/aesthetics/`**: Want to know the "soul" of each country? Here you'll find the moodboards, sensations, and vibe checks for places like Greece, France, Italy, and more.
- **`/docs/visual-identity/`**: The hardcore design rules. Here you'll find the exact color palettes, typography pairs, and layout rules we use to keep things looking premium but culturally adapted.
- **`/docs/shared/`**: The base styles that *never* change (grid, spacing, UI components) to keep the app feeling like one unified platform.
- **`/docs/interaction-design/`**: The "Anti-Noise Manifesto". It explains how we use animations (GSAP, Lenis, Framer Motion) to make things feel buttery smooth without looking like a chaotic laser show.

## 🛠️ Tech Stack & Running the App

We are keeping things smooth, fast, and modern:
- **Next.js (App Router)** & **TypeScript**
- **Tailwind CSS v4** (say goodbye to massive config files!)
- **shadcn/ui** (customized to look ultra-minimalist)
- **GSAP & Lenis** for that heavy, high-end magazine scroll feel.

**To run the project locally:**
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) and enjoy the view!

## Contributions & Forking

I hope you like this project! Feel free to contribute, fork it, or just steal some ideas for your own stuff. 

**If you want to contribute:**
1. **Fork the repo** and make it your own for a bit.
2. **Branch out**: `git checkout -b feature/your-awesome-idea`
3. **Keep it chill**: Try to follow the design rules in the `docs/` folder so we don't break the "Silent Premium" vibe. If you add animations or components, remember: *less is more*.
4. **Push & PR**: Send a Pull Request explaining what you did and why it's cool. I'd love to see it!

Oh, and if you find a bug, just open an Issue. No stress, we fix it together.
