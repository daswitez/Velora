# Typography: Main Landing Page

## The Global Voice
Because the Main Landing operates strictly above the country level, it must not lean into any specific nation's typographic identity. It uses the pure, unadulterated Velora brand fonts.

## Typeface Selection
- **Display / Headers:** `Playfair Display` (Elegant, authoritative, editorial).
- **Body / Functional / UI:** `Inter` (Precise, invisible, highly legible).

## Hierarchical Rules
1. **Hero Title (`H1`):** `Playfair Display`, Light or Regular. Massive size (e.g., `text-6xl md:text-8xl`). Leading tightly controlled (`leading-[0.9]`).
2. **Subheadline:** `Inter`, Light. Small, spaced out (`tracking-widest uppercase`).
3. **Body Copy:** `Inter`, Regular. Neutral sizing (`text-base` to `text-lg`). Maximum width set (`max-w-[65ch]`) to ensure perfect editorial line lengths.
4. **Interactive Accents (Links, Buttons):** `Inter`, Medium. Small text (`text-sm`), often paired with an ultra-thin 1px underline or an arrow icon (`Lucide/Phosphor`).

## Avoidance
- Do not use any of the specific country Display fonts (like the German Industrial Sans or the Greek Classicist Serif) on this page to maintain neutrality.
