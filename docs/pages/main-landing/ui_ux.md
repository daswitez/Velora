# UI/UX: Main Landing Page (Diplomatic Headquarters)

## The Core Objective
The Main Landing serves as the "Diplomatic Headquarters". Its primary UX goal is **elegant routing**. It does not sell a specific property; it sells the *concept* of Velora and acts as a sophisticated gateway to the 8 "Country Embassies".

## Key Components

1. **The Grand Hero (Gateway)**
   - **UX:** Minimal interaction required. A massive, high-quality architectural video or image. 
   - **UI:** A very clean, centered typography block introducing Velora, with a subtle scroll indicator.

2. **The "Select Your Destination" Map / Carousel**
   - **UX:** The user needs to clearly see the 8 available markets. Hovering should reveal hints of that country's aesthetic.
   - **UI:** A horizontal, buttery-smooth Embla Carousel or an interactive list. Clicking a country acts as a portal, smoothly transitioning the user to that specific Embassy.

3. **The AI "Concierge" Intro**
   - **UX:** Introduce the NLP Smart Search feature before they even reach a country. "Tell us how you want to live."
   - **UI:** A minimalist, glowing input field. It shouldn't look like a standard web form, but rather a luxurious search terminal.

## Navigation Context
- The Global Navbar here is in its "Neutral" state. It does not adopt any country's specific color palette yet. It remains strictly black, white, and translucent glass.
