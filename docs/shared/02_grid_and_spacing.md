# Grid & Spacing (Air)

## Overview
White space is not a void in VELORA Living; it is a design material in its own right. Contemporary luxury is transmitted through the "air" that allows every element of the interface to breathe.

## The Core Grid
- **Structure:** A classic 12-column system adapted for ultra-wide resolutions (max-width spanning 1440px and beyond, compensated with fluid margins).
- **Ample Proportions:** Unlike mass-market real estate portals that attempt to compress dozens of listings into the viewport, VELORA allows each element, photograph, and text block to occupy generous proportions.

## The Concept of "Silent Premium" Spacing
- **Breathing Zones:** The internal padding of master sections exceeds usual standards (e.g., jumping to `py-24` or `py-32` in Tailwind).
- **Editorial Rhythm:** Scrolling down the page, especially on the "Diplomatic Headquarters" (Main Landing), assumes a relaxed reading pace. The user scrolls into laid-back compositions where every short paragraph is given ample distance from the edges and adjacent content.
- **Unexpected Alignments:** While hard data (filters, prices) is rigidly aligned (mathematical order), editorial blocks (neighborhood descriptions or lifestyle notes) can afford subtle asymmetrical alignments to reinforce the perception of an art magazine.

## Image Handling
- **Bleed:** Hero photographs frequently break the grid to go full-bleed (full-width), stretching from edge to edge for maximum visual impact, setting the narrative state before demanding interaction.
