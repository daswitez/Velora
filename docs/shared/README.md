# Shared Styles - VELORA Living

This directory houses the structural and genetic foundation of **VELORA Living**. While the `aesthetics` folder governs the cultural and individual pigmentation of each country, this folder contains the immutable laws of the system.

The directives documented here form the **Design System Root** that both the Main Landing and every single country experience will inherit. It is the "common language" that allows the brand to feel European, unified, supremely premium, hyper-legible, and international.

## The 4 Unshakable Pillars

1. **[Mother Typography](01_typography.md)**  
   *The reading rhythm and the elegant collision of high editorial (Serif) with technical precision (Sans-Serif).*

2. **[Grid and Spacing (Air)](02_grid_and_spacing.md)**  
   *The art of composing with "air". Massive proportions, deep margins, and asymmetrical magazine rhythms.*

3. **[Base Components and UI](03_ui_components.md)**  
   *Global sober navigation, the NLP search engines, majestic property cards, and silky micro-interactions.*

4. **[Tone of Voice](04_editorial_tone.md)**  
   *The vocabulary and style manual. Communicating cultural intelligence without ostentation (Silent Premium).*
