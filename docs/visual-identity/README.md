# Intelligent Visual Identity - VELORA Living

This directory documents the core of VELORA's **Adaptive Design System**. It is the logical framework that allows the platform to transcend being a single, rigid European identity, acting instead as an organism that mutates its visual temperature based on the culture of the page/country the user visits.

## The Challenge: Respecting Identity and Aesthetics
Achieving institutional cohesion without destroying the local soul ("Silent Premium" at scale).

### Identity Documentation

1. **[01_adaptability_rules.md](01_adaptability_rules.md)**  
   *The borders of the system. Which elements (grids, pure usability) are constitutionally closed to changes, and which elements (color, hero typographies, visual climate) are 100% fluid.*

2. **[02_typography_by_country.md](02_typography_by_country.md)**  
   *The linguistic engine. The architectural assignment of Display typography to each country based on its geographical character and design history (e.g., Modernist Helvetica in Amsterdam vs. warm Humanist Serif in Italy).*

3. **[03_color_palettes.md](03_color_palettes.md)**  
   *The 8 color palettes applied through UX hierarchy (Base, Surface, Text, Accent) that absorb everything from historical local pantones to climate-driven light temperatures.*

4. **[04_page_layouts.md](04_page_layouts.md)**  
   *The structural dissection of a VELORA Embassy. How we mathematically squeeze the 12-column grids for the Hero, the Smart Search (NLP Assistant), the high-end Gallery, and the Editorial Zone to guarantee the "Premium Magazine" status.*
